# 🚫 YouTube EGE School Blocklist

<img width="1728" height="818" alt="photo_2026-05-30_02-51-27" src="https://github.com/user-attachments/assets/50173c86-ac0e-463c-904a-171cba630794" />

Список каналов российских онлайн-школ (Умскул, Школково, Фоксфорд и т. п.), которые навязчиво продвигаются в рекомендациях YouTube.  
Предназначен для использования с блокировщиками рекламы, поддерживающими косметические правила — например, [uBlock Origin](https://ublockorigin.com).

## Зачем?

Автор считает, что видео с этих каналов являются скорее развлекательным, чем образовательным контентом, а следовательно — бесполезными. Таких каналов становится всё больше, и этот список призван убрать их из рекомендаций, чтобы было легче находить нормальные образовательные каналы.

## Готовый список
Скопировать [output.txt](https://github.com/kiberdans/youtube-ege-blocklist/blob/main/output.txt)

## Установка фильтра (uBlock Origin)

1. Откройте **Панель управления** uBlock Origin.
2. Перейдите на вкладку **Мои фильтры**.
3. Вставьте содержимое `output.txt` (или `filter.txt` из [releases](https://github.com/kiberdans/youtube-ege-blocklist/releases)).
4. Нажмите **✓ Применить изменения**.

## Приложение (GUI) для создания списков

В репозитории есть Python-приложение (`main.py`) на `customtkinter`, которое позволяет:

| Возможность | Описание |
|---|---|
| Загрузка списка | Читает ссылки/хендлы из `links.txt` |
| Фильтрация | Поиск по названию, кнопки «все / выбраные / невыбраные» |
| Drag-to-toggle | Зажал ЛКМ и провёл по списку — чекбоксы переключаются |
| Контекстное меню | ПКМ: копировать handle, открыть канал в браузере |
| Двойной клик | Открывает канал на YouTube с подсветкой |
| Сохранение | Генерирует `output.txt` с правилами для uBlock Origin |
| Состояние | `state.json` сохраняет отметки между запусками |

### Установка и запуск GUI

**Linux:**
```bash
bash install_linux.sh
python3 main.py
```

**Windows (PowerShell):**
```powershell
.\install_windows.ps1
python main.py
```

## Создание собственного списка
- Копируем ссылки на ютуб каналы. К примеру https://www.youtube.com/@umschool (просто пример любого канала, все совпадения случайны)
- Вставляем все каналы в `links.txt`
- Заходим в GUI
- Можем редактировать галками каналы которые попадут в финальный блеклист
- Нажимаем кнопку `сохранить правила`
- Открываем `output.txt`, копируем всё и вставляем в uBlock


## Как это работает

Фильтр использует косметические правила, которые находят в HTML-коде страницы ссылки на конкретные YouTube-каналы (например, `/@umsch_phys`) и полностью скрывают соответствующие элементы.  
Список построен на основе CSS-селекторов с псевдоклассом `:has()`, поэтому обрабатывается браузером максимально быстро.

Пример правила:
```
youtube.com##ytd-video-renderer:has(a[href*="/@umsch_phys"], a[href*="/@shkolkovo"])
```

Оно скрывает любое видео, в карточке которого есть ссылка на один из перечисленных каналов.
