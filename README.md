# 🚫 YouTube EGE School Blocklist

Список каналов российских онлайн-школ (Умскул, Школково, EXtremum, Вебиум и др.), которые навязчиво продвигаются в рекомендациях YouTube.  
Предназначен для использования с блокировщиками рекламы, поддерживающими косметические правила к примеру [uBlock Origin](https://ublockorigin.com).

## Зачем?

Автор считает что данные видео являются развлекательным контентом больше чем образовательным, в следствии бесполезным контентом. Но так как подобных каналов всё больше и больше, этот список призван удалить подобные каналы, что бы стало легче искать нормальные каналы.

## Установка

uBlock Origin
1. Откройте **Панель управления** uBlock Origin.
2. Перейдите на вкладку **Мои фильтры**.
3. Вставьте содержимое `filter.txt`.
4. Нажмите **✓ Применить изменения**.

## Как это работает

Фильтр использует косметические правила, которые находят в HTML-коде страницы ссылки на конкретные YouTube-каналы (например, `/@umsch_phys`) и полностью скрывают соответствующие элементы.  
Список построен на основе оптимизированных CSS-селекторов с псевдоклассом `:has()`, поэтому обрабатывается браузером максимально быстро.

Пример правила: `youtube.com##ytd-video-renderer:has(a[href="/@umsch_phys"], a[href="/@shkolkovo_phys"]`

Оно скрывает любое видео, в карточке которого есть ссылка на один из перечисленных каналов.
